# Pigeon: Drone

## What Pigeon Is?
Pigeon is a autopilot drone which works on Algorithms &amp; CV to capture terrain and find spots for landing like a moon lander! Also the drone can be switched to manual at any moment, although it will be only used in emergency situations or maybe only in prototyping stage. I am thinking to use a mini SBC in drone which can process the images from camera mounted on drone using AI libraries like PyTorch to find spots for landing!

## What Pigeon Does?
Pigeon is an automated drone, which can take a flight by its own and use AI algorithms to find landing spots, also it can return to the user in one click by using the location sensor. Also I am thinking to mount some sensors to get environment+terrain data which can be wirelessly send over LoRa communication. 

## Why I build Pigeon?
I always wanted to make a drone which is fully autonomous or require partial human interference to track it. Pigeon is just not for my personal experience+fun but also it can have some applications that can help the environment, especially tracking the pollution, moisture, temperature etc. Also pigeon can become a data carry package which can store data in air and directly communicate to ground wirelessly. Also I find it really cool if a drone has its own brain how will it predict patterns in the terrain and how it can find landing spots in a rugged training.
